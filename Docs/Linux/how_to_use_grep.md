```
find . -type f -path "*/config/ファイル名" -exec grep -n "検索文字列" {} +
```

コマンドの意味

- 1 find .:　find はファイルやディレクトリを再帰的に検索するコマンドです。
　　　　　　. は現在のディレクトリを意味します。これにより、カレントディレクトリ以下を再帰的に検索します。

- 2 -type f:　-type f は、通常のファイルのみを検索対象とするオプションです。ディレクトリやその他のタイプのファイル（シンボリックリンクなど）は除外されます。

- 3 -path "*/config/ファイル名":　-path は検索するファイルのパスを指定するオプションです。*/ は、任意のサブディレクトリのパスに一致します。

- 4 -exec grep -n "max_viewpoint" {} +: -exec は、find で見つけた各ファイルに対してコマンドを実行するオプションです。

grep -n "検索文字列" {} は、見つけた各ファイルに対して grep コマンドを実行し、その中で max_viewpoint という文字列を検索します。
grep は指定された文字列を検索するコマンドです。
-n オプションは、見つかった文字列が何行目にあるかを表示します。
{} は find で見つけたファイル名に置き換えられます。
+ は -exec オプションにおいて、できるだけ多くのファイルを一度に grep コマンドに渡すことを意味します。これにより、複数のファイルを効率的に検索することができます。
