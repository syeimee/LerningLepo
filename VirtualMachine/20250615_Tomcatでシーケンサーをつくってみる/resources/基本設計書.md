# 概要
Tomcatでシーケンサーをつくってみる

# ディレクトリ構成
```bash
/WebContent
   ├─ /sounds               ← 静的音声ファイル置き場（今回は任意）
   ├─ /WEB-INF
   │     ├─ web.xml
   │     └─ /jsp
   │          ├─ login.jsp
   │          ├─ sequencer.jsp
   └─ /js
         └─ sequencer.js
/src
   └─ /servlet
         ├─ LoginServlet.java
         ├─ LogoutServlet.java
         └─ SequencerServlet.java

```
