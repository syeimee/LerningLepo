## 2.メッセージ一覧

エラーコードは以下のフォーマットで作成すること

- 1桁目
  - E : エラー
  - W : ワーニング
- 2〜8桁目
  - APIのIDの数字7桁
- 9〜12桁目
  - 0埋めした任意の4桁の数字
  - ただし、以下の番号は予約番号とする


|コード|内容|返却メッセージ|備考|使用API|
|--------|--|--|--|--|
|9901|予期しないエラー|予期しないエラー|想定していない汎用的（ぬるぽ等）なエラー|全API共通|
|9902|DBエラー|DBエラー|DBの通信エラーや一意制約違反といったDB関連のエラー|全API共通|
|9903|外部サービス通信エラー|外部サービス通信エラー|宅配サービスのAPIやクレジットのAPIで出たエラー|全API共通|
|0001|バリデーションエラー|バリデーションエラー|リクエストパラメータの型や値が不正|全API共通|

エラーコードは4桁ごとにハイフンでつなぐこと

記入例(ID0010002で発生した予期せぬエラー)
E001-0002-9901
