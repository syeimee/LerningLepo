## アウトプットパラメータの形式について
各APIのアウトプットパラメータは、以下の形式とし、各章に記載されたパラメータはparamsに設定されるものとする
|レベル|論理名|物理名|型|サイズ|空条件|備考|
|-----|-----|-----|-----|-----|-----|-----|
|1|ステータス|status|Integer|-|-|0:正常終了 10:警告終了 20:異常終了|
|1|エラーコード|errCd|String|14|正常終了時|XXXX-XXXX-XXXX形式|
|1|エラーメッセージ|errMsg|String|0-n|正常終了時|-|
|1|パラメータ|params|Object|0-1||-|
