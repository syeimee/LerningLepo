# 学習テーマ
作業日時: 2025-07-21


## 目的・背景 
AIによるvibe コーディングが流行っており、中でもClaudeCodeは最近注目されている。
今回はClaudeCodeを使って簡単なアプリを作ってみることにする。


## 実装内容・学んだ技術  
### 1. Claude Codeのインストール

```bash
npm install -g @anthropic-ai/claude-code
```
※Mac環境は上記コマンドで問題なくインストールできるはずだが、Win環境はLinux環境を用意する必要があるかも。


### 2.起動
```bash
claude
```
